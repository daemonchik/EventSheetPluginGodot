🎮 EventSheet Plugin

Визуальное событийно-ориентированное программирование для Godot 4

✅

v1.0 - Основы

Завершено Production Ready

Полнофункциональная система событийно-ориентированного программирования с базовым функционалом для создания игровой логики без написания кода.

📦 Включает:

✓ Runtime система
✓ Визуальный редактор
✓ События (Input, Physics)
✓ Действия (Objects, Physics)
✓ Условия (Comparisons)
✓ Система выражений
✓ JSON сохранение
✓ Плагинная система
